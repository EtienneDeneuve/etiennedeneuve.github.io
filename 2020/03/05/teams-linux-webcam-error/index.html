Your webcam is mirrored in Microsoft Teams in Linux ?

Teams on Linux

I use Teams on my linux laptop to work on Ansible and other open source python project.

Today, I checked if my webcam is properly displayed in Teams.
My webcam was reversed.

Fix it

Edit the your shortcut :

code /usr/share/applications/teams.desktop

and change the line Exec by this one :

Exec=sh -c 'export LIBV4LCONTROL_FLAGS=1 && LD_PRELOAD=/usr/lib/x86_64-linux-gnu/libv4l/v4l1compat.so usr/bin/teams %U'

You might check where is v4l1compat.so on your system using :

locate v4l1compat.so
/usr/lib/x86_64-linux-gnu/libv4l/v4l1compat.so
